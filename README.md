# DAS Group 09 – Group Project 2

## Overview
This project analyses Dataset 9 from the IMDB film database. The aim of the project is to investigate which properties of films influence whether a film receives an IMDB rating greater than 7.

## Research Question
Which properties of films influence whether they are rated by IMDB as greater than 7?

## Dataset
The dataset contains information about films such as year of release, film length, budget, number of votes, genre and the IMDB rating.

## Method
The analysis is carried out in R using a Generalised Linear Model (GLM). Since the response variable is whether the rating is greater than 7 or not, a logistic regression model is used.
